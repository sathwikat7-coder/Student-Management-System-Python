# Student Management System (Python)

This is a console-based Python project developed to manage student records efficiently.

## Description
The Student Management System allows users to add, view, update, and delete student details. It uses file handling to store data, ensuring records are saved even after the program is closed.

## Features
- Add new student records
- View all students
- Update existing student details
- Delete student records
- Persistent data storage using text files

## Technologies Used
- Python

## How to Run the Project
1. Download or clone the repository
2. Run the program using:
   python student_management.py
